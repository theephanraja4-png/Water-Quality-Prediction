# 💧 Water Quality Prediction

A Machine Learning web application built using **Streamlit** that predicts whether water is safe for drinking based on various water quality parameters.

## 🚀 Features

* Interactive Streamlit interface
* Real-time water quality prediction
* Machine Learning classification model
* Simple and responsive UI
* Fast and reliable predictions

## 🛠️ Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Joblib

## 📁 Project Structure

```text
Water_Quality_Prediction/
│── app.py
│── water_potability.csv
│── water_model.pkl
│── requirements.txt
│── README.md
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/theephanraja4-png/Water-Quality-Prediction.git
```

Navigate to the project folder:

```bash
cd Water-Quality-Prediction
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

## 📊 Input Features

* pH
* Hardness
* Solids
* Chloramines
* Sulfate
* Conductivity
* Organic Carbon
* Trihalomethanes
* Turbidity

## 🎯 Output

* Safe for Drinking
* Not Safe for Drinking
